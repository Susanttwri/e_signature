# eSignSure - Modern eSignature Platform

eSignSure is a fast, secure, and legally-binding online eSignature software. This project has been transformed into a professional multi-page application with a built-in blog CMS.

## Features

- **Multi-Page Layout**: Professional "About Us", "FAQ", and "Resources" pages.
- **Blog CMS**: Integrated Admin Panel to manage blog posts and resources.
- **Interactive UI**: Responsive design with smooth animations and square blog card layouts.
- **Legally Binding**: Optimized for document signature workflows.

## Pages

- **Home**: Main landing page with hero banner and benefits.
- **About Us**: Information about our mission and impact.
- **FAQ**: Frequently asked questions about security and usage.
- **Resources**: A hub for blog posts and essential guides.
- **Admin**: Internal panel for creating and managing content.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) to see the site.

## Technologies Used

- **Next.js 16**: App Router and Server Components.
- **React 19**: Modern UI library.
- **React Bootstrap**: Responsive styling components.
- **FontAwesome**: Professional iconography.
- **CSS Modules & Global Styles**: Custom interactive styling.
